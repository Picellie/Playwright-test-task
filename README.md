**PLAYWRIGHT TEST TASK**

This project contains an automated test written using Playwright.

**The test:**

1. Opens the website https://playwright.dev/
2. Verifies that the page contains full Playwright title.
3. Runs in multiple browsers:

- Chromium
- Firefox

Other browsers can be added in playwright.config.ts

You can check the status of the automated tests in the [Actions](https://github.com/Picellie/Playwright-test-task/actions) tab on GitHub.
